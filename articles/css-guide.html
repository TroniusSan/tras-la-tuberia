CSS Guide - Styling Web Pages

Beginner Level • 8 min read

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and positioning of HTML elements.

Three Ways to Add CSS

1. Inline CSS

<p style="color: blue; font-size: 18px;">Styled paragraph</p>

2. Internal CSS

<style>
  p { color: blue; }
</style>

3. External CSS (Recommended)

<link rel="stylesheet" href="styles.css">

CSS Selectors

Element Selector

p { color: blue; }

Class Selector

.highlight { background-color: yellow; }

ID Selector

#header { font-size: 2rem; }

The Box Model

Every element in CSS has a box model consisting of:

  • Content - The actual content
  • Padding - Space around content
  • Border - Surrounds the padding
  • Margin - Space outside the border
.box {
  margin: 10px;
  border: 2px solid black;
  padding: 15px;
}

Flexbox Layout

Flexbox is a powerful tool for creating flexible layouts:

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}

Responsive Design

Use media queries to adapt your design for different screen sizes:

@media (max-width: 768px) {
  .container { flex-direction: column; }
}

Common CSS Properties

  • color - Text color
  • background-color - Background color
  • font-size - Size of text
  • width, height - Element dimensions
  • margin, padding - Spacing
  • border - Element border
  • border-radius - Rounded corners

Key Takeaways

  • CSS makes web pages look beautiful and professional
  • Use external CSS files for better organization
  • Understand the box model for proper spacing
  • Flexbox and Grid are essential for modern layouts
  • Always test your design on mobile devices
