CSS Guide

Intermediate Level • 8 min read

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, and positioning of HTML elements.

How to Apply CSS

1. Inline CSS

<p style="color: red;">This text is red</p>

2. Internal CSS

<style>
p { color: blue; }
</style>

3. External CSS

<link rel="stylesheet" href="styles.css">

CSS Selectors

  • Element Selector: p { }
  • Class Selector: .className { }
  • ID Selector: #idName { }
  • Universal Selector: * { }
  • Attribute Selector: input[type="text"] { }

Box Model

Every element consists of:

  • Content - The actual content inside the element
  • Padding - Space inside the element border
  • Border - Line around the padding
  • Margin - Space outside the border
.box {
  margin: 10px;
  padding: 20px;
  border: 1px solid black;
}

Flexbox Layout

Flexbox makes it easy to arrange elements in a row or column:

.container {
  display: flex;
  justify-content: center;
  gap: 20px;
}

Grid Layout

CSS Grid is perfect for creating complex layouts:

.grid {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 15px;
}

Responsive Design

Use media queries to make your site responsive:

@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}

Common Properties

  • color: Text color
  • background-color: Background color
  • font-size: Text size
  • width/height: Element dimensions
  • opacity: Transparency (0-1)
  • transition: Smooth animations

Key Takeaways

  • CSS controls the visual appearance of HTML elements
  • Use external stylesheets for maintainability
  • Master the box model for better layouts
  • Flexbox and Grid are modern layout solutions
  • Always design with mobile responsiveness in mind
