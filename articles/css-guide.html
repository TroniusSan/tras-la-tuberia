🎨 CSS Guide

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to control fonts, colors, sizes, spacing, and add animations. CSS separates styling from HTML structure.

CSS Selectors

CSS selectors target HTML elements to apply styles:

/* Element Selector */
p { color: blue; }

/* Class Selector */
.highlight { background-color: yellow; }

/* ID Selector */
#header { font-size: 2rem; }

The Box Model

Every element has the box model consisting of:

  • Content - The actual content
  • Padding - Space inside the element
  • Border - Surrounds the padding
  • Margin - Space outside the border
.box {
  margin: 10px;
  border: 2px solid black;
  padding: 15px;
}

Flexbox Layout

Flexbox is a powerful layout tool for creating flexible designs:

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  gap: 10px;
}

Grid Layout

CSS Grid provides a 2D layout system:

.grid {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 20px;
}

Responsive Design

Use media queries for different screen sizes:

@media (max-width: 768px) {
  .container {
    flex-direction: column;
  }
}

Common CSS Properties

  • color - Text color
  • background-color - Background color
  • font-size - Size of text
  • width / height - Dimensions
  • border - Element border
  • border-radius - Rounded corners
  • box-shadow - Shadow effect
  • transition - Smooth animations

💡 Key Takeaways

  • CSS styles HTML elements beautifully
  • Learn selectors, box model, and layouts
  • Flexbox and Grid are essential modern tools
  • Always design with responsive design in mind
  • Practice styling different elements!
