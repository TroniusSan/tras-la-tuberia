📝 HTML Basics

What is HTML?

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It provides the structure and content of websites. HTML uses a system of tags and elements to tell web browsers how to display content.

Basic HTML Structure

Every HTML document starts with a basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Page Title</title>
</head>
<body>
  <h1>Welcome!</h1>
</body>
</html>

Common HTML Tags

  • <h1> to <h6> - Headings (h1 is the largest)
  • <p> - Paragraph of text
  • <a> - Hyperlink to another page
  • <img> - Embed an image
  • <div> - Container for grouping content
  • <ul> / <ol> - Unordered and ordered lists
  • <li> - List item
  • <button> - Interactive button
  • <form> - Input form for user data

Semantic HTML

Semantic HTML uses meaningful tags that describe the content. This improves accessibility and SEO. Examples include:

<header> - Page header
<nav> - Navigation links
<main> - Main content
<section> - Thematic grouping
<article> - Self-contained content
<footer> - Page footer

Creating Your First Page

Follow these steps:

  1. Create a new file and save it as index.html
  2. Add the basic HTML structure
  3. Add your content using appropriate tags
  4. Save the file and open it in a browser

💡 Key Takeaways

  • HTML provides the structure of web pages
  • Tags come in pairs: opening <tag> and closing </tag>
  • Use semantic tags for better accessibility
  • DOCTYPE declares the HTML version
  • Practice creating pages to master HTML!
