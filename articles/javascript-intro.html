JavaScript Introduction

Beginner Level • 10 min read

What is JavaScript?

JavaScript is a programming language that makes web pages interactive. It runs in the browser and allows you to respond to user actions.

Where to Write JavaScript

1. Inline JavaScript

<button onclick="alert('Hello!')">Click me</button>

2. Script Tag

<script>
console.log('Hello World');
</script>

3. External File

<script src="script.js"></script>

Variables

Variables store data that you can use and manipulate:

let name = "John";
const age = 25;
var city = "New York";
  • let: Block-scoped, preferred choice
  • const: Cannot be reassigned, use by default
  • var: Avoid using (outdated)

Data Types

  • String: Text data - "Hello"
  • Number: Integer or decimal - 42, 3.14
  • Boolean: True or false
  • Array: List of values - [1, 2, 3]
  • Object: Collection of properties - {name: "John"}

Functions

Functions are reusable blocks of code:

function greet(name) {
  return "Hello, " + name;
}

greet("Alice");

Arrow Functions

A modern way to write functions:

const add = (a, b) => a + b;
add(5, 3); // Returns 8

Control Flow

If Statement

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

Loops

for (let i = 0; i < 5; i++) {
  console.log(i);
}

DOM Manipulation

Select and modify HTML elements:

// Select element
const btn = document.getElementById("myBtn");

// Change content
btn.textContent = "Click me!";

// Add event listener
btn.addEventListener("click", () => {
  alert("Button clicked!");
});

Array Methods

  • map(): Transform array elements
  • filter(): Keep elements that match condition
  • forEach(): Loop through array
  • find(): Find first matching element
  • push(): Add element to end

Key Takeaways

  • JavaScript adds interactivity to web pages
  • Use const by default, let when needed
  • Functions help organize and reuse code
  • DOM manipulation allows you to change HTML dynamically
  • Array methods like map and filter are powerful tools
  • Always use the browser console for debugging
