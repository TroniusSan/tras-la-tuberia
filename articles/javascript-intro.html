JavaScript Introduction

Beginner Level • 10 min read

What is JavaScript?

JavaScript is a programming language that makes web pages interactive. It runs in your browser and allows you to add interactivity, animations, and dynamic content.

Variables and Data Types

Variables store data values. Use const, let, or var to declare them:

const name = "John"; // String
const age = 25; // Number
const isStudent = true; // Boolean
const hobbies = ["reading", "coding"]; // Array

Functions

Functions are reusable blocks of code:

function greet(name) {
  return `Hello, ${name}!`;
}

greet("Alice"); // Output: Hello, Alice!

DOM Manipulation

JavaScript can interact with HTML elements using the DOM (Document Object Model):

// Select element
const button = document.getElementById("myButton");

// Change text
button.textContent = "Click me!";

// Add event listener
button.addEventListener("click", function() {
  alert("Button clicked!");
});

Events

JavaScript responds to user interactions:

  • click - Mouse click
  • mouseover - Mouse hovers over element
  • change - Form input changed
  • keydown - Key pressed
  • submit - Form submitted

Conditional Statements

Control program flow with if/else:

const score = 85;

if (score >= 90) {
  console.log("Grade: A");
} else if (score >= 80) {
  console.log("Grade: B");
} else {
  console.log("Grade: C");
}

Loops

Repeat code multiple times:

// For loop
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// While loop
let count = 0;
while (count < 3) {
  console.log(count);
  count++;
}

Arrays and Objects

Arrays

const fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // apple
fruits.push("grape"); // Add item

Objects

const person = {
  name: "John",
  age: 30,
  city: "New York"
};
console.log(person.name); // John

Key Takeaways

  • JavaScript adds interactivity to web pages
  • Use variables to store data
  • Functions organize reusable code
  • DOM manipulation changes HTML dynamically
  • Events respond to user interactions
  • Practice by building interactive projects!
